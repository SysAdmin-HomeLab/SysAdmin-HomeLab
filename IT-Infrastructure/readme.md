# IT Infrastructure Labs

This repository contains hands-on lab work focused on IT support, system administration, networking, databases, cloud concepts, and cybersecurity fundamentals.

The goal is to build practical, real-world technical skills through structured home lab environments and documentation.


## 📌 Repository Structure

- **Linux** – System administration, commands, services, and troubleshooting
- **Windows Server** – Active Directory, DNS, DHCP, and Windows administration
- **Networking** – CCNA fundamentals, subnetting, routing, and Packet Tracer labs
- **IT Support** – Hardware, OS troubleshooting, and helpdesk-style scenarios
- **Security** – Basic security concepts, system hardening, and monitoring
- **Cloud** – Cloud computing fundamentals (AWS/Azure concepts and labs)
- **Database** – Basic SQL, database setup, and administration concepts


## 🎯 Learning Goals

- Develop strong IT support and troubleshooting skills
- Build system administration experience (Linux & Windows)
- Understand networking fundamentals (CCNA-level knowledge)
- Gain exposure to cloud and security concepts
- Prepare for IT support and system administrator roles
- Transition into cybersecurity in the future


## 🛠️ Tools & Technologies

- Linux (Ubuntu/Debian)
- Windows Server
- Cisco Packet Tracer
- VirtualBox / Home Lab Environments
- Basic SQL (MySQL/PostgreSQL)
- Git & GitHub


## 📈 Approach

All work in this repository is based on:
- Hands-on practice
- Home lab environments
- Step-by-step documentation
- Troubleshooting and real-world scenarios


## 🚀 Purpose

This repository serves as a personal IT infrastructure portfolio demonstrating practical skills across multiple IT domains.
